# Movie Explorer

Just another flutter project that uses omdbAPI to fetch movies.

## Preview

## Get Compiled APK

## Dependencies
``` yaml
  provider: ^5.0.0
  http: ^0.13.2
  ```

## Features
  * HTTP API Request
  * Error Handling
  * Item grid listing 
  * Add items to favorites
  * User confirmation popups
